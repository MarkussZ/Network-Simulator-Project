# Network Simulator

A Python-based network simulator designed to allow users to practice configuring network devices and connections in a simulated environment.

## Technologies
Python  
MySQL  
Matplotlib  

## Features
- Create and connect different network devices
- Configure IP addresses, subnets, and gateways
- Store network configuration data in a MySQL database
- Visualise the network topology using graph-based visualisation

## Future Improvements
- Interactive graphical interface for dragging and configuring devices
- Web-based deployment to allow remote access
- Additional device types and configuration options
